# Lukulu Recordings Brand Identity

## Core visual direction

Lukulu Recordings should look like an Afro House record label, not a generic software dashboard.

The uploaded reference set defines the brand language:

- carved wood textures
- metallic silver and gold treatments
- black studio/nightclub contrast
- African geometric pattern engraving
- map-of-Africa emblem
- speaker cones
- traditional drum details
- hand-crafted Lukulu wordmark energy
- scratched, embossed, tactile surfaces
- colorful African textile artwork variants for social and release campaigns

## Primary UI treatment

Use this direction for Lukulu Hub:

- deep black background
- charcoal and dark brown texture
- metallic silver highlights
- muted gold accent states
- subtle scratches and paper/wood grain
- embossed Afro-pattern panels
- rounded premium cards
- release artwork tiles that feel like record sleeves
- speaker/waveform details for music context

## Brand modes

### 1. Premium metallic mode

Best for dashboard, analytics, executive view, app shell, and internal tools.

- black / charcoal base
- silver or gold logo treatment
- low-saturation lighting
- embossed patterns
- subtle scratches

### 2. Wood-carved heritage mode

Best for label profile, biography, artist profile, press kit, and cultural storytelling.

- warm brown wood
- carved Africa shape
- traditional instrument references
- organic shadows
- handcrafted look

### 3. Color textile mode

Best for social posts, campaign moments, artist spotlights, and celebratory releases.

- red, green, yellow, teal, and black pattern fields
- high-energy African textile background
- gold/cream Lukulu wordmark
- dense geometry behind release artwork

## Implementation notes

- Avoid flat SaaS styling.
- Avoid generic purple gradients unless used very sparingly.
- Use patterns as texture, not as distracting wallpaper.
- Keep text readable over dark backgrounds.
- Use approval-first workflows for generated social content.
- Never store social passwords; use Meta OAuth only.
